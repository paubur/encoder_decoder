# encoder_decoder

App to encode and decode text.

encoder:
- encode text to format: separator, encoded_text, seprator, list_of_original_words

decoder:
- decode encoded text to format of original one (delivered text must be encoded)

# To start the APP
```
Open your terminal
$ git clone https://github.com/paubur/encoder_decoder
$ cd encoder_decoder
Run tests:
$ python3 -m unittest
To run terminal version of encoder type:
$ python3 script.py
```
